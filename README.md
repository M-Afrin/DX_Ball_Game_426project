# DX Ball Game 🎮

A DX Ball-inspired brick breaker game developed in **C++ using OpenGL/GLUT** as part of the **CSE 426 Graphics Laboratory** course project.

The game recreates the classic brick-breaking gameplay where the player controls a paddle to bounce a ball and destroy bricks while trying to survive with limited lives.

---

## Features

* Interactive Main Menu
* Multiple Difficulty Levels (Easy, Medium, Hard)
* Sound Effects and Background Music Support
* High Score Saving System
* Life Tracking System
* Dense Triangular Brick Layout
* Keyboard Controlled Paddle Movement
* Game Over and Player Name Entry Screen
* Credits Screen

---

## Technologies Used

* C++
* OpenGL
* GLUT / FreeGLUT
* Code::Blocks
* Windows Multimedia API (PlaySound)

---

## Controls

| Key         | Action            |
| ----------- | ----------------- |
| Left Arrow  | Move Paddle Left  |
| Right Arrow | Move Paddle Right |
| Spacebar    | Launch Ball       |
| Mouse Click | Navigate Menu     |
